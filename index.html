Introduction

CSS (Cascading Style Sheets) is the code that styles web content.

It is used to style and layout web pages â€” for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

What is CSS?

Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is what you use to selectively style HTML elements. For example, this CSS selects paragraph text, setting the color to blue:

p { color: blue; }

When a browser reads a style sheet, it will format the HTML document according to the information in the style sheet.

There are 3 ways to insesrt CSS:

  • Inline styles are CSS declarations that affect a single HTML element, contained within a style attribute.
    <p style="color:red;"> This is a paragraph.</p>

  • Internal stylesheet may be used if one single HTML page has a unique style. The internal style is defined inside the <style> element, inside the head section.
    <style> h1 { text-transform: capitalize; } </style>

  • External stylesheet contains CSS in a separate file with a .css extension. This is the most common and useful method of bringing CSS to a document. You can link a single CSS file to multiple web pages, styling all of them with the same CSS stylesheet.
    <link rel="stylesheet" href="styles.css" />

CSS Syntax

A CSS ruleset consists of a selector and a declaration block.

The selector points to the HTML element you want to style.

The declaration block contains one or more declarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.

Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

In this example, p is a selector in CSS (it points to the HTML element you want to style). While color is a property, and red is the property value

CSS Selectors

CSS selectors are used to select the HTML elements you want to style.

Type of selectors:

  • Type selector selects HTML elements based on the element name <h1>
  • The class selector selects HTML elements with a specific class attribute .container
  • The id selector uses the id attribute of an HTML element to select a specific element #main-doc
  • The universal selector * selects all HTML elements on the page.
  • Attribute selectors select elements based on an attribute or attribute value a[target="_blank"]
  • Pseudo-classes style certain states of an element eg. when the user hovers over an element :hover
  • Pseudo-elements select a certain part of an element rather than the element itself ::first-line
